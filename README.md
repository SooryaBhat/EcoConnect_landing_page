# 📌 About

EcoConnect-X is an IoT-enabled smart waste management ecosystem integrating:
- ESP32 smart bins
- Flask backend
- MySQL database
- Real-time monitoring dashboards
- Recycling workflows
- GIS monitoring support
- Multilingual accessibility

This landing page was created to professionally showcase the complete project architecture, workflows, dashboards, and smart-city monitoring features.
