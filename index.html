About the Project

What is EcoConnect-X?

An engineering project that turns everyday dustbins into data-connected smart bins and ties the entire waste ecosystem into one web platform.

EcoConnect-X is an IoT-based waste management system built for urban and semi-urban municipalities. Smart dustbins equipped with ESP32 microcontrollers and load cell sensors measure waste weight and push live data to a Flask backend over WiFi.

The platform connects four types of users — citizens, waste collectors, recyclers, and government authorities — each with a dedicated dashboard tailored to their role. Citizens report complaints and sell recyclable waste. Collectors track daily tasks and bin fill levels. Recyclers browse the waste marketplace. Government monitors regional analytics and manages bin requests.

The system is designed to be affordable and practically deployable — built on standard hardware and a lightweight Python stack, with no complex infrastructure needed.

How the hardware works

Each smart bin has a load cell mounted at its base, connected to an HX711 amplifier that converts the analog weight reading into a digital signal. An ESP32 microcontroller reads this signal and sends the fill-level data to the Flask backend via WiFi at regular intervals.

When a bin crosses a fill threshold, the system automatically flags it on the collector dashboard and triggers an alert — no manual checking needed.

Language Support

Three Languages, One Platform

EcoConnect-X supports multilingual dashboards for better accessibility across local communities in coastal Karnataka.

English Interface

The default platform language. All dashboards — citizen, collector, recycler, and government — are available in English with clear labels, status indicators, and action buttons.
