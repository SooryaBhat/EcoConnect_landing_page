About the Project

What is EcoConnect-X?

EcoConnect-X is an IoT-based waste management platform built for urban and semi-urban municipalities. Smart dustbins equipped with ESP32 microcontrollers and load cells measure waste weight and transmit live data to a Flask backend over WiFi.

The platform connects four stakeholders — citizens, collectors, recyclers, and government — through dedicated role-based dashboards. Citizens report complaints and sell recyclable waste. Collectors track bin fill levels and daily tasks. Recyclers browse the waste marketplace. Government authorities monitor regional analytics and bin requests.

Built as a practical engineering project, EcoConnect-X demonstrates how affordable IoT hardware combined with a clean web backend can bring real visibility to a city's waste ecosystem.

The Smart Bin Hardware

Each smart bin is fitted with a load cell connected to an HX711 amplifier and an ESP32 microcontroller. The ESP32 reads waste weight continuously and pushes that data over WiFi to the Flask backend at regular intervals.

The hardware is low-cost, compact, and designed to fit inside a standard dustbin — practical for real-world municipal deployment without heavy infrastructure.

Language Support

Three Languages, One Platform

EcoConnect-X supports English, Kannada, and Tulu — the three primary languages spoken in coastal Karnataka. This makes the platform usable by a much wider range of citizens, including those not comfortable with English-only interfaces.

Language accessibility is a core design decision. All key workflows — complaints, waste listing, and bin requests — are available in the local language.
